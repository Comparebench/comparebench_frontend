Roadmap

We've put together a very basic roadmap so you can see what we're working on and where our priorities are for each "milestone". We don't have an ETA for these items, but we'll keep this and our twitter updated as we build our platform.

Milestone 1

Early items that need some extra polish

  • Allow users to edit Benchmark Profiles and Comparisons
  • Add more benchmarks to an existing Profile
  • Allow users to search through their benchmarks and comparisons
  • Infrastructure additions

Milestone 2

Quality-of-life additions for the project

  • Add photos to benchmark profiles
  • Allow users to compare their benchmarks with others
  • Flesh out product pages with average score results
  • Add more support for other CPU and GPU benchmarks
  • Add more support for gathering system information

Milestone 3

Final feature-sets

  • Allow users to embed comparison graphs elsewhere
  • Full Leaderboard system
  • Public Results API
  • Benchmark submission API
  • Determine product performance based on user-submitted results
